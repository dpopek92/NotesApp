## Description

## Installation

```bash
# from main dir

# installing dependencies for frontend & backend
$ npm run postinstall

# installing dependencies to run the project
$ npm i
```

## Running the app

```bash
# from main dir

# run project in dev mode
$ npm run dev
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```

## Backend API

```bash
# when project is running

# Open API
$ http://localhost:8000/api
```
