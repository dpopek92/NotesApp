## Description

## Installation

```bash
# from main dir

# installation dependencies for frontend & backend
$ npm run postinstall

# installation dependencies for running the project
$ npm i
```

## Running the app

```bash
# from main dir

# run project in dev mode
$ npm run dev
```

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```

## Backend API

```bash
# when project is running

# Open API
$ http://localhost:8000/api
```
